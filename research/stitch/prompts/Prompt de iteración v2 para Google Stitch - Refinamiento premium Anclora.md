# Prompt de iteración v2 para Google Stitch - Refinamiento premium Anclora

Este prompt sirve para una segunda iteración en `Google Stitch` cuando ya existe una primera propuesta visual y hace falta elevarla desde una dirección editorial premium hacia un sistema reusable más sólido para el ecosistema Anclora.

## Cuándo usarlo

- cuando `Google Stitch` ya ha generado una primera versión aceptable pero demasiado específica o poco reusable
- cuando la propuesta sigue pareciendo demasiado `showcase` y no suficiente `sistema`
- cuando hace falta exigir una `dark version` realmente diseñada
- cuando la composición mobile pierde tensión o jerarquía

## Prompt principal

```text
Refine the current landing into a stronger Anclora premium system template.

This should no longer feel like a luxury real-estate showcase only. It must become a reusable premium landing skeleton that can later adapt to AI products, editorial platforms, data products, advisory services, or hybrid digital experiences across the Anclora ecosystem.

Keep the current strengths:
- editorial clarity
- quiet luxury
- European premium digital product feeling
- serif + sans contrast
- restrained sophistication
- generous spacing
- high-trust visual tone

But improve the design substantially with the following direction:

SYSTEM-LEVEL IMPROVEMENTS
- Shift the design from “brand showcase” to “reusable premium product landing system”.
- Make the section logic more modular and reusable across multiple app types.
- Strengthen the product-story architecture so the page can explain software, service, or hybrid offers, not only aspirational advisory.
- Preserve a premium tone, but make the structure feel more adaptable and operational.

LAYOUT AND COMPOSITION
- Increase intentional asymmetry in the hero and in at least 2 additional sections.
- Break away from safe stacked blocks and introduce more editorial composition.
- Create stronger focal rests and better rhythm between dense and quiet areas.
- Reduce sections that feel like default template rows.
- Improve transitions between sections so the page feels curated, not assembled.

VISUAL LANGUAGE
- Replace visible divider-line logic with tonal layering, spacing, background shifts, and surface hierarchy.
- Reduce generic web-card and shadow aesthetics.
- Push the visual system toward layered paper, soft glass, ambient depth, and architectural framing.
- Keep the look premium, calm, and highly designed.
- Avoid startup clichés, generic luxury clichés, purple SaaS gradients, noisy overlays, and dashboard aesthetics.

COMPONENT REFINEMENT
- Make CTA treatment more distinctive and premium.
- Keep only one dominant CTA per main viewport or section.
- Redesign FAQ so it feels authored and premium, not like a default accordion list.
- Improve proof/trust sections so they feel intentional, not placeholder logo rows.
- Make the footer feel like a composed final brand space, not a standard multi-column footer.
- Ensure cards and content modules feel elegant and reusable, not locked to one domain.

MOBILE IMPROVEMENTS
- Rework the mobile layout to preserve impact.
- Compress excessive vertical spacing where it weakens momentum.
- Keep strong visual anchors on small screens.
- Make sure each section still has a focal point in mobile.
- Simplify or restack low-value compositions that become flat in narrow viewports.
- Mobile should feel designed, not just responsively collapsed.

DARK MODE
- Create a true dark-mode version designed from first principles.
- Do not simply invert the light theme.
- Use a premium dark palette built from deep teal, navy, warm near-black, muted gold highlights, and soft readable text tones.
- Dark mode should feel atmospheric, editorial, calm, luxurious, and precise.
- Preserve hierarchy, contrast, and compositional richness in dark mode.
- Ensure the dark version feels equally premium and intentional, not secondary.

ANCLORA PREMIUM IDENTITY
- The result should feel like a signature Anclora premium landing system.
- It should feel suitable for a family of sophisticated digital products.
- It should combine elegance, clarity, conversion structure, and modularity.
- It must not feel like a one-off visual concept tied only to architecture or real estate.

DELIVER
- an improved light version
- a deliberately designed dark version
- clearer modular section logic
- stronger reusable product-landing structure
- more distinctive Anclora premium identity
- tighter, more impactful mobile layout

Important:
- Keep the page premium but restrained.
- Make it feel more system-level, more reusable, and more compositionally sophisticated.
- Prioritize hierarchy, rhythm, adaptability, and premium visual discipline.
```

## Variante corta de follow-up

```text
Push this design from premium editorial showcase into a reusable Anclora landing system. Increase asymmetry, improve modular product storytelling, remove divider-line logic, replace generic cards/shadows with tonal layering, redesign mobile spacing and focal points, create a truly composed dark mode, and make the CTA, FAQ, proof sections, and footer feel more distinctive, premium, and reusable across AI, editorial, data, and advisory apps.
```
