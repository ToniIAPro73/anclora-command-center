# Prompt de consolidación canónica para Google Stitch - Sistema premium Anclora

Este prompt está pensado para el momento en que `Google Stitch` ya ha explorado suficientes variantes y deja de tener sentido seguir abriendo ramas visuales nuevas. Su función es obligar a Stitch a converger hacia un único sistema premium Anclora, coherente entre `light` y `dark`, reusable y canónico.

## Cuándo usarlo

- cuando ya existen varias exploraciones y hace falta elegir
- cuando el sistema visual empieza a fragmentarse entre variantes paralelas
- cuando se necesita pasar de exploración a consolidación
- cuando la prioridad deja de ser descubrir direcciones y pasa a ser fijar una gramática reusable

## Prompt principal

```text
Consolidate the current Stitch explorations into one canonical Anclora premium landing system.

You have already explored multiple directions, including warmer editorial “atelier” variants and darker atmospheric “nocturne” variants. Stop branching. Do not generate another parallel concept line. Synthesize the best parts into a single, coherent, reusable Anclora system.

Goal:
Create one canonical premium landing design language for Anclora that can scale across current and future ecosystem apps.

This system must:
- feel premium, editorial, calm, sophisticated, and high-trust
- work as a reusable landing skeleton, not a one-off showcase
- support AI, editorial, data, advisory, and hybrid product narratives
- exist as one family across light and dark mode
- feel distinctly Anclora, not generic SaaS, not generic luxury, not just architectural real estate

What to do now:

1. CONVERGE THE VISUAL DIRECTIONS
- Merge the strongest qualities of the “atelier” and “nocturne” explorations.
- Keep the editorial elegance, asymmetry, and breathable composition from the lighter direction.
- Keep the atmospheric depth, tonal sophistication, and premium dark-mode thinking from the darker direction.
- Produce one unified design language, not two separate aesthetics.

2. DEFINE A CANONICAL SYSTEM
Decide and lock:
- the primary visual mood
- the core light palette
- the core dark palette
- the typographic hierarchy
- the surface hierarchy
- the CTA logic
- the section rhythm
- the image treatment
- the motion attitude

Do not keep multiple competing interpretations. Choose one system.

3. REMOVE INTERNAL CONTRADICTIONS
- If the system says “no divider lines,” do not use visible divider-line layouts.
- If the system says “tonal layering,” make that visible in the actual composition.
- If asymmetry is part of the concept, apply it intentionally in layout, not only in theory.
- Ensure the visual output matches the design rationale.

4. PRIORITIZE REUSABILITY
The landing should no longer feel tied mainly to luxury real estate or architectural advisory.
It must feel adaptable to:
- premium AI applications
- editorial tools
- data products
- high-end advisory services
- ecosystem platforms

Design the page as a premium modular framework with reusable blocks.

5. ESTABLISH MODULAR BLOCKS
Create a canonical landing structure with reusable sections such as:
- hero
- value proposition
- trust / authority
- product or service narrative
- proof / outcomes
- FAQ
- final CTA
- footer

Each section should feel like a premium module that can later be adapted by product type without breaking the system.

6. LIGHT AND DARK AS ONE FAMILY
- Create a light version and a dark version that clearly belong to the same design family.
- Dark mode must not be an inversion.
- Light mode must not feel flat or generic.
- Both must preserve the same hierarchy, cadence, identity, and component logic.

7. MOBILE MUST BE DESIGNED, NOT COLLAPSED
- Tighten spacing where mobile becomes too passive.
- Preserve focal points.
- Keep premium rhythm on narrow screens.
- Ensure modular sections still feel intentional in mobile.

8. KEEP THE ANCLORA PREMIUM IDENTITY
The final result should feel like:
- European editorial digital product
- quiet luxury
- precision without sterility
- premium but restrained
- modern but not trend-chasing
- modular but not generic

Avoid:
- generic SaaS blocks
- luxury clichés
- dashboard aesthetics
- decorative excess
- disconnected concept branches

Deliver:
- one canonical light direction
- one canonical dark direction
- one unified Anclora premium landing system
- clearer modular section logic
- a more reusable product-landing architecture
- stronger consistency between design rationale and output
```
