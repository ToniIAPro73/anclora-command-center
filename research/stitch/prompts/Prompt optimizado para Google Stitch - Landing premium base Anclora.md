# Prompt optimizado para Google Stitch - Landing premium base Anclora

Este prompt está pensado para `Google Stitch`, con instrucciones más directas y orientadas a generación visual rápida. Parte del mismo sistema premium de Anclora, pero reduce ambigüedad y verbosity para obtener una primera propuesta de layout más usable.

## Cuándo usarlo

- cuando se quiera sacar una primera dirección visual rápida en `Google Stitch`
- cuando haga falta un prototipo visual base antes de iterar copy o estructura
- cuando se busque una landing premium reusable sin entrar aún en detalle técnico

## Prompt para Google Stitch

```text
Design a premium reusable landing page for Anclora Group that can serve as the base template for any current or future app in the ecosystem.

The page should feel editorial, sophisticated, modern, calm, high-trust, and conversion-oriented. It must not look like a generic SaaS landing page or an internal dashboard with cosmetic styling.

Creative direction:
- premium European digital product
- quiet luxury
- editorial clarity
- refined technology
- elegant hierarchy
- strong first impression without visual noise

Visual principles:
- premium layout with strong rhythm and generous spacing
- one dominant CTA in the hero
- subtle gradients, texture, layered backgrounds, soft glass only if controlled
- refined cards and surfaces with clear separation
- sophisticated typography with personality and strong readability
- premium buttons with confident presence, not aggressive
- visual depth without clutter
- no startup clichés, no purple SaaS aesthetic, no noisy UI

Motion direction:
- subtle fade
- short rise
- brief stagger
- calm premium transitions
- no bounce
- no theatrical animations

Structure:
- premium hero with immediate clarity
- value proposition block
- authority / trust layer
- problem or opportunity framing
- modular solution narrative
- product or experience showcase
- proof / credibility section
- FAQ
- final CTA
- premium footer

Design constraints:
- desktop and mobile friendly
- modular and reusable
- easy to adapt later to AI, data, editorial, services, or platform products
- premium but restrained
- clear conversion hierarchy
- polished, intentional, scalable

Optional brand direction:
Use a premium palette that can support deep teal, textured gold, navy, warm neutrals, or other restrained luxury tones depending on the target app.

Output goal:
Create a high-end landing page skeleton with premium brand atmosphere, clear conversion structure, modular sections, and strong visual identity suitable for the Anclora ecosystem.
```

## Variante corta

```text
Create a premium modular landing page for Anclora Group. The design should feel editorial, sophisticated, high-trust, modern, and quietly luxurious. Build a reusable landing skeleton for future ecosystem apps, with a strong hero, one dominant CTA, elegant hierarchy, refined cards, subtle depth, premium typography, calm motion, trust sections, FAQ, and final conversion block. Avoid generic SaaS patterns, startup clichés, purple gradients, or dashboard aesthetics. Make it polished, adaptable, and premium on desktop and mobile.
```
