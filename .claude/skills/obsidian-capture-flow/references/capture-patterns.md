# Capture patterns

Prefer these choices:

- Inbox note: use for unfinished thoughts, meeting scraps, copy-pasted chat fragments, and links that still need judgment.
- Daily note: use for items anchored to a date, such as work logs, retrospectives, and end-of-day summaries.
- Resource note: use for source material that is already useful on its own after a light cleanup.

Practical heuristics:

- If the note would otherwise sit unopened for days, send it to `00 Inbox`.
- If time is part of the meaning, send it to `01 Diario`.
- If someone should be able to find and reuse it later without reprocessing, send it to `04 Recursos`.

Keep capture notes short. A rough note with provenance is more valuable than a polished note with uncertain origin.
