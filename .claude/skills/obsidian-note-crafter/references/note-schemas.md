Use these section shapes as defaults, not rules.

```markdown
---
tipo: proyecto
estado: activo
creado: 2026-03-26
---

# Nombre del proyecto

Resumen breve.

## Objetivo

## Estado actual

## Próxima acción

## Relacionado
```

```markdown
---
tipo: recurso
fuente:
```
