# Review checklist

Check for:

- Titles that only make sense in chat context
- Notes with no links and no clear reason to exist alone
- Overuse of tags where links would be stronger
- Missing source fields on imported or clipped material
- Mixed property names for the same concept such as `estado`, `status`, or `situacion`
- Project notes with no explicit next action
- Inbox notes that should be promoted, merged, or archived

Good cleanup is boring, safe, and consistent. Avoid dramatic vault redesign unless the user asks for it.
