# CLAUDE.md

Este archivo se mantiene por compatibilidad con flujos y agentes que esperan `CLAUDE.md`.

La guía canónica y neutral para este repositorio es `AGENTS.md`.

## Compatibilidad
* Si un agente usa `CLAUDE.md` como memoria o instrucciones del proyecto, debe tratar este archivo como un puente hacia `AGENTS.md`.
* Las reglas de estructura de la bóveda, uso de skills y criterios editoriales viven en `AGENTS.md`.
* La fuente canónica de skills del repo es `.codex/skills/`; `.claude/skills/` debe mantenerse sincronizada desde ahí.
