# Anclora Private Estates Landing Page

## 📋 Descripción

Repositorio operativo dedicado a la landing pública de [[Anclora Private Estates]]. Su función no es comportarse como catálogo masivo ni como dashboard reetiquetado, sino como superficie `flagship` de autoridad, posicionamiento territorial, captación cualificada y acceso selectivo al ecosistema del vertical.

## 🏁 Resultado esperado

Disponer de una landing ultra premium alineada con el sistema canónico de Anclora y capaz de sostener:

- autoridad editorial en Palma y el suroeste de Mallorca
- captación de propietarios, compradores e inversores premium
- derivación selectiva hacia `Data Lab` y `Synergi`
- expansión multilenguaje `es/en/de/fr`

## 🎯 Objetivos

- Objetivo principal: construir la puerta pública premium del vertical inmobiliario sin depender de un stock amplio de inmuebles.
- Objetivo secundario: traducir el canon `dark/light` de Anclora a una experiencia ultra premium específica para real estate.
- Objetivo secundario: dejar una base técnica y documental clara para evolucionar la landing hacia producto real sin rehacer su estructura.

## 📍 Estado actual

- Situación actual: repo inicializado y estructurado como base `Vite + React + TypeScript`.
- Último avance relevante: documentación estratégica consolidada a partir de conversaciones con ChatGPT y Grok, sistema canónico `Stitch` y contratos universales + ultra premium.
- Principal riesgo o bloqueo: todavía no existe implementación visual completa ni validación real de `dark/light`, i18n y overrides ultra premium.

## 🧾 Marco contractual

- Familia contractual: `Ultra Premium`.
- Idioma objetivo: `es/en/de/fr`.
- Tema objetivo: `premium multi-theme`, con prioridad de diseño en `dark` y traducción deliberada a `light`.
- Contratos aplicables: [[ANCLORA_ECOSYSTEM_CONTRACT_GROUPS]], [[ANCLORA_ULTRA_PREMIUM_APP_CONTRACT]], [[ANCLORA_PREMIUM_APP_CONTRACT]], [[UI_MOTION_CONTRACT]], [[MODAL_CONTRACT]], [[LOCALIZATION_CONTRACT]].
- Criterio de diseño: la landing debe cumplir primero el canon de [[Sistema canónico de landing premium Anclora]] y después adaptarlo al vertical `Private Estates`.

## 🧱 Base técnica actual

- Base en `Vite + React + TypeScript`.
- Estructura separada en `docs/`, `public/`, `src/` y `tests/`.
- Preparación explícita para `content`, `i18n`, `seo`, `sections` y assets territoriales.
- Repo remoto asociado en GitHub y listo para implementación incremental.
