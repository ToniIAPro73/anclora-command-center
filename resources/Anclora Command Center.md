# Anclora Command Center

`Anclora Command Center` es la superficie ejecutiva del ecosistema Anclora. Su función es condensar señal, prioridades, alertas, dependencias y estado operativo en una capa de lectura rápida para decisión y seguimiento.

## Qué es

No es el lugar donde nace la señal ni donde se ejecuta el trabajo de detalle. Actúa como capa de síntesis para leer el sistema y decidir foco.

## Rol en el ecosistema

- Lee señal operativa desde [[Anclora Nexus]].
- Hace visible la inteligencia que llega desde [[Anclora Data Lab]].
- Ayuda a seguir activaciones derivadas de [[Anclora Content Generator AI]].
- Da contexto ejecutivo a la capa relacional de [[Anclora Synergi]].

## Subsuperficies relevantes

- [[Anclora Cuadro de Mando Real Estate]] como vista específica del dataset Real Estate.

## Criterio de uso

- Usar esta nota como nodo canónico cuando el foco sea la superficie ejecutiva general.
- Usar [[Anclora Cuadro de Mando Real Estate]] cuando el foco sea la lectura específica del vertical inmobiliario.
