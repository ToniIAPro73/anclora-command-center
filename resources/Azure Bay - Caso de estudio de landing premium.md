# Azure Bay - Caso de estudio de landing premium

## Resumen

`Azure Bay` es un caso de estudio ficticio usado como portfolio para demostrar cómo Toni plantea una landing inmobiliaria premium con estructura de conversión, copy bilingüe y narrativa orientada a inversión.

El repositorio local está en:

- `C:\Users\antonio.ballesterosa\Desktop\Proyectos\anclora-azure-bay-landing`

## Qué demuestra

- diseño visual premium aplicado a real estate
- estructura comercial de captación
- narrativa de inversión en español e inglés
- integración pensada para formularios y automatización

## Stack visible

Según el `README` del repo:

- Next.js 16
- TypeScript
- Tailwind CSS
- ALTCHA
- HubSpot Forms API

## Valor narrativo

Este activo sirve para mostrar:

- sensibilidad estética
- capacidad de posicionar un producto de ticket alto
- criterio para ordenar una landing que no parezca genérica
