# Anclora Portfolio - Base técnica reutilizable

## Resumen

`Anclora Portfolio` es la base técnica reusable que mejor demuestra cómo Toni estructura un proyecto digital para real estate premium con ambición de escalar, no solo de verse bien.

El repositorio local está en:

- `C:\Users\antonio.ballesterosa\Desktop\Proyectos\anclora-portfolio`

## Qué demuestra

- arquitectura modular mantenible
- frontend premium con criterio de marca
- captura de contactos con validación
- rate limiting
- persistencia
- calidad técnica suficiente para evolucionar a producto real

## Stack visible

Según el `README` del repo:

- Next.js 16
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui + Radix UI
- Framer Motion
- Zod
- Vitest

Además, deja preparadas dependencias e infraestructura para mayor complejidad:

- Prisma
- NextAuth
- TanStack Query
- Zustand

## Valor narrativo

Este activo es especialmente útil para explicar que Toni no solo puede encargar o prototipar una landing, sino construir una base de producto reutilizable para nuevos clientes o nuevas promociones.

La frase importante aquí no es “sé maquetar”, sino:

- sé diseñar una base técnica limpia para captar, validar, almacenar y evolucionar un sistema comercial digital
