# Ficha de producto - Anclora Azure Bay Landing

`Azure Bay` es un caso de estudio de landing premium para Real Estate que sirve como plantilla viva de storytelling, estructura de conversión y look & feel de activo prime.

## Vista usuario

- Qué es: landing ficticia premium para una promoción inmobiliaria de alto nivel.
- Para quién es: clientes, partners y equipo interno como demostración de cómo se vería una campaña digital premium.
- Propuesta de valor: muestra cómo empaquetar narrativa, inversión y conversión en una superficie de lujo.
- Casos de uso: demo comercial; referencia de diseño; base para futuras landings reales.
