# Defuddle Reference

## Typical usage
```bash
npx defuddle parse https://example.com/article --md
npx defuddle parse https://example.com/article --json
```

## Best uses
* clean article capture
* provenance-preserving web clipping
* reference-note ingestion

## Practical rules
* Preserve source URL and date.
* Prefer JSON output when metadata matters.
* Use a custom selector only when the default extraction fails.
