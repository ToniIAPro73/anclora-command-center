# Flujo Comercial Inteligente Anclora

## 🎯 Objetivo

Describir la integracion entre datos, contenido, pipeline comercial y partnerships para que el vertical inmobiliario de Anclora opere como un sistema coordinado y no como herramientas aisladas.

## 🧭 Cuándo usarlo

- Cuando haya que convertir senales de mercado en accion comercial
- Cuando un lead premium necesite una narrativa o activo especifico para avanzar
- Cuando un partner deba recibir contexto, materiales o seguimiento estructurado

## ✅ Precondiciones

- [[Anclora Data Lab]] tiene identificadas senales territoriales, criterios o insights relevantes
- [[Anclora Nexus]] dispone de etapas de pipeline visibles
- [[Anclora Synergi]] puede asociar partners o workspaces a oportunidades concretas

## 🪜 Pasos

1. Tomar como input las senales territoriales e insights de [[Anclora Data Lab]].
2. Traducir esas senales en briefs, emails segmentados o activos comerciales desde [[Anclora Content Generator AI]].
3. Activar esos activos dentro de [[Anclora Nexus]] segun la etapa del lead o la oportunidad.
4. Usar [[Anclora Synergi]] para onboarding de partners, coordinacion privada y continuidad relacional.
5. Registrar feedback comercial para refinar futuros contenidos, senales y criterios de activacion.

## 🔎 Validación

- Cada pieza de contenido se puede rastrear a una senal o insight concreto
- Las oportunidades en [[Anclora Nexus]] muestran que activo se ha usado y con que resultado
- Los partners reciben materiales consistentes con el contexto de negocio

## ⚠️ Riesgos y rollback

- Riesgo principal: generar contenido desconectado del pipeline o de la inteligencia real
- Cómo deshacer o corregir: volver a mapear senal -> contenido -> activacion -> resultado y eliminar pasos ambiguos

## Definición del Flujo

Este playbook describe la integracion de datos y contenido para el vertical inmobiliario.

- Input: senales territoriales de [[Anclora Data Lab]].
- Procesamiento: [[Anclora Content Generator AI]] genera briefs de captacion y emails segmentados.
- Activación: [[Anclora Nexus]] distribuye el contenido segun la etapa del lead.
- Relación: [[Anclora Synergi]] utiliza estos activos para el onboarding de partners.
