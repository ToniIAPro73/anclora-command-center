# Borrador de Email de Captación Synergi

## Contexto

Borrador inicial basado en el flujo `dato -> contenido -> activacion -> relacion`. El mensaje toma como referencia un perfil de partner premium como [[Lucia Serrano]] y usa como gancho la promesa de acceso filtrado, admision privada e inteligencia territorial de [[Anclora Data Lab]].

## Borrador

Asunto: Acceso privado a inteligencia territorial y oportunidades filtradas

Hola Lucia,

Estoy impulsando [[Anclora Synergi]] como entorno privado para partners que necesitan algo mas que un flujo comercial generico: acceso filtrado, trazabilidad y contexto real antes de dedicar tiempo a una oportunidad.

Lo estamos conectando con [[Anclora Data Lab]], una capa de inteligencia pensada para detectar senales territoriales, validar interes y convertir esa informacion en materiales accionables para colaboradores seleccionados.

La idea no es enviarte volumen, sino acceso a un workspace privado donde podamos compartir:

- oportunidades con mejor encaje para tu perfil
- criterios de admision y seguimiento claros
- insights territoriales relevantes antes de mover una conversacion comercial

Si te encaja, puedo enviarte un primer resumen de enfoque y explicarte como funcionaria el acceso privado para partners.

Un abrazo,

Antonio Ballesterosa  
[[personas/Toni|Toni]]  
[[Anclora Group]]

## Variables para adaptar

- Tipo de partner: inversor, broker, developer o colaborador comercial
- Insight o senal concreta: territorio, demanda, ticket medio, perfil de activo
- CTA: llamada corta, acceso a workspace o envio de resumen inicial
